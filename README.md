# IA Master OS

Aplicação Next.js moderno com TypeScript, Tailwind CSS, ESLint e App Router.

## 🚀 Stack de Tecnologias

- **Next.js 14** - Framework React com App Router
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Utility-first CSS framework
- **ESLint** - Linter para código JavaScript/TypeScript
- **Turbopack** - Bundler rápido

## 📦 Configurações

✅ TypeScript  
✅ ESLint  
✅ Tailwind CSS  
✅ App Router  
✅ Turbopack  
✅ Import Alias (@/*)  

## 🏃 Como Começar

### Pré-requisitos

- Node.js 16+
- npm ou yarn

### Instalação

```bash
# Clone o repositório
git clone https://github.com/duranabner/ia-master-os.git
cd ia-master-os

# Instale as dependências
npm install
```

### Desenvolvimento

```bash
# Inicie o servidor de desenvolvimento
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

### Build para Produção

```bash
# Build da aplicação
npm run build

# Inicie o servidor de produção
npm start
```

### Linting

```bash
# Verifique o código
npm run lint
```
